About

Halo, saya Anggit. Saya adalah seseorang yang memiliki hobi menggambar, khususnya pada bidang hand lettering, yaitu seni typography yang biasanya dibuat di atas kertas. Hobi ini bermula ketika saya duduk di bangku SMP. Saya melihat teman saya jago dalam melakukan hand lettering sehingga membuat saya tertarik dalam dunia hand lettering ini. Dari situ saya mulai belajar dari teman saya dan dengan meniru serta memodifikasi karya orang lain sampai akhirnya saya bisa membuat banyak karya sendiri. Lalu saya memiliki niat untuk mempublikasikan karya saya, akhirnya saya memilih instagram sebagai medianya dan saya membuat akun instagram dengan username SiluetType untuk mempublikasikan karya saya.
